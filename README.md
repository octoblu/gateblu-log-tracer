# gateblu-log-tracer
Trace what happened to that dog gone gateblu device

```text
Usage: gateblu-log-tracer [options] [command]


 Commands:

   list-activity      get list of gateblu activity in last 24 hours
   list-failures      get list of gateblu-deployment failures in last 24 hours
   group-failures     get list of gateblu failed activity grouped by gateblu in last 24 hours
   trace              trace a gateblu deployment
   gateblu-activity   show activity for a gateblu uuid
   help [cmd]         display help for [cmd]

 Options:

   -h, --help     output usage information
   -V, --version  output the version number
```
